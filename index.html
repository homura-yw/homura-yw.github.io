由一道算法题所启发的井字棋游戏

算法 2022/8/23

缘由

  • 几个月前参加学校举办的算法比赛时，遇见过一个难度颇为不易的题目，题目大致内容是通过输入一个井字棋的棋盘，然后求某方要赢的情况的种数。虽然井字棋本身非常的简单，但要用程序模拟出下棋的情况总数，终归还是难到我了。虽然但是，我通过不断递归遍历的算法，终究还是把这道题做出来了。
  • 这道题用递归能适用，很大一部分在于，井字棋本身是非常简单的，总共不过3*3=9个棋子，考虑到棋盘上已经有棋子了，所以每种情况递归的层数都不会超出8，这些为递归提供了条件。
  • 将这道题水过去后，我一直感觉，本题在某种程度上几乎就是让我写了一个预测所有情况的AI，将所有的情况综合考虑，即可得出，在走 剩余格子 中的一个后，所获胜的种数，所失败的种数以及评价的种数，进而可以求得走这一步获胜的概率！而只要每一步都走获胜概率最高的一步，这不就形成了一个可以自己下棋的AI了吗？理论成立，实践的种子也埋在了我的心里，也正是这样促使我完成一个简单的井字棋游戏。

    建立一个游戏的操作框架

    为了减少程序的耦合度，我将程序分为两个模块，一个是游戏本身的操作框架模块，另一个是游戏的AI内核模块（说人话就是这一模块是来选择走哪一步棋的）。

显然，游戏本身操作框架模块是非常easy的。由于本人不会c++的前端设计，所以直接通过小黑框，进行标准输入和输出。

游戏AI内核的模块

在上面，有一个非常关键的函数

void putPiece(char qipan[][3])

这个函数的作用便是，传入棋盘的数组后，将决定走下一步棋的位置，让我们看看在”井.h”中的putPiece函数是怎么写的：

void putPiece(char qipan[][3]) {
    void Copy(char qipan[][3], char Qipan[][3]);
    int stepin(char qipan[][3], int cnt, int step, int W);

    int cnt = 0;
    for (int I = 0; I < 3; I++) {
        for (int i = 0; i < 3; i++) {
            if (qipan[I][i] == '_') cnt++;
        }
    }
    char Qipan[3][3] = { 0 };
    double gailv[9] = { 0 };
    int top = 0;
    for (int I = 0; I < 3; I++) {
        for (int i = 0; i < 3; i++) {
            if (qipan[I][i] == '_') {
                Copy(qipan, Qipan);
                Qipan[I][i] = 'O';
                double X_win = stepin(Qipan, cnt - 1, 1, 1);
                double O_win = stepin(Qipan, cnt - 1, 1, 2);
                double ping = stepin(Qipan, cnt - 1, 1, 0);
                gailv[top++] = (ping + O_win) / (ping + O_win + X_win);
            }
        }
    }
    double max = 0;
    int num = 0;
    for (int i = 0; i < top; i++) {
        if (gailv[i] >= max) {
            max = gailv[i];
            num = i;
        }
    }
    int count = 0;
    for (int I = 0; I < 3; I++) {
        for (int i = 0; i < 3; i++) {
            if (qipan[I][i] == '_') {
                count++;
            }
            if (count == num + 1) {
                qipan[I][i] = 'O';
                return;
            }
        }
    }
}

这其中有两个函数

void Copy(char qipan[][3], char Qipan[][3]);

int stepin(char qipan[][3], int cnt, int step, int W);

其中Copy结合上下文不难理解，这个其实就是一个简单的复制棋盘数组的代码
真正重要的函数便是这个stepin，这也是整个程序不断递归的函数（同时参数也多得离谱）
stepin函数的作用，就是返回对应结局的种数，这里先贴一下代码，不然之后讲起来就有点混乱,因为这个函数还是有点复杂的

int stepin(char qipan[][3], int cnt, int step, int W) {
    bool isnotInline(char qipan[][3]);
    int whowin(char qipan[][3]);
    void Copy(char qipan[][3], char Qipan[][3]);
    void set(char* site, int step);

    if (whowin(qipan) != 0) {
        if (whowin(qipan) == W) return 1;
        else return 0;
    }

    if (cnt == 1) {

        int right = 1;
        int wrong = 0;

        char Qipan[3][3];
        Copy(qipan, Qipan);
        char* p;
        for (int I = 0; I < 3; I++) {

            for (int i = 0; i < 3; i++) {
                if (Qipan[I][i] == '_') {
                    p = &Qipan[I][i];
                    set(p, step);
                    goto end;
                }
            }
        }
    end:
        if (W == whowin(Qipan)) {

            return right;

        }
        else return wrong;

    }

    

    int sum = 0;
    for (int J = 0; J < cnt; J++) {
        char QiPan[3][3];
        Copy(qipan, QiPan);

        int j = 0;
        char* p;
        for (int I = 0; I < 3; I++) {

            for (int i = 0; i < 3; i++) {
                if (QiPan[I][i] == '_') {
                    p = &QiPan[I][i];
                    j++;
                    if (j > J) {
                        set(p, step);
                        goto END;
                    }
                }
            }
        }
    END:
        sum += stepin(QiPan, cnt - 1, step + 1, W);
    }
    return sum;
}

这个函数中，参数cnt代表这个棋盘还剩下的棋子数，step代表当前已走的棋子数（有一说一，这俩其实知道一个就行了，主要是这是全盘抄的比赛时的代码，当时没有多想，现在又不敢乱动这个代码了，因为调试起来真的会死人），至于这个W，则是选定的要赢的棋子（或者更妥当一点，应该是选择X或O赢或平局）。其中还得简要提一嘴这个函数

int whowin(char qipan[][3])

这个就是判断棋子胜负的函数，返回1代表X赢，返回2代表O赢，返回0代表平局或者棋盘未满。当

if (whowin(qipan) == W)

成立时，stepin停止递归，并返回1，当不成立且w!=0时，返回0

对stepin函数的总结

stepin函数继续递归的条件:

  • 无任意一方棋子连为3个

  • cnt值不为0
    其中，最后一个for循环就是对于走每一个空格的满足参数W情况的平行递归的总和，即：

    END:
      sum += stepin(QiPan, cnt - 1, step + 1, W);
  }
  return sum;
